# WorkloadIndicator

## Installation

### Dependencies
- Arduino library and ide in the folder "C:\Program Files (x86)\Arduino"

### Including libraries
- Adafruit Neopixel library [Github](https://github.com/adafruit/Adafruit_NeoPixel) to control the ws2812b
